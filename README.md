## 这是个啥？

简单说，这是一个帮你做**潜意识音频**的工具。

你有没有听过"潜意识编程"？就是把自己的目标、愿望录成音频，但处理成"听不见"的高频声音，混在背景音乐里。你的耳朵听不到，但大脑能接收到。

**举个例子：**
- 你想变得更自信
- 用手机录一句"我越来越自信了"
- 用这个工具处理一下
- 配上冥想音乐
- 每天睡前听30分钟
- 坚持66天，潜移默化

## 怎么用？

### 方式一：在线使用（最简单）

直接打开：**https://subliminal-audio-synthesizer.onrender.com**

1. 下载项目
2. 双击 `启动Subliminal_Master.bat`

### 操作步骤

```
第一步：上传音频
├── Track A: 上传肯定句录音（支持多选批量处理）
└── Track B: 上传背景音乐

第二步：调整参数
├── 选择预设（默认/深度冥想/专注学习/助眠放松）

第三步：开始合成
├── 点击"开始合成"
├── 等待处理完成（有提示音）
└── 在线试听 / 下载

第四步：保存使用
├── 下载WAV文件
└── 每天听1-2次，坚持66天
```

## 参数怎么调？

### 快速选择预设

| 预设 | 适用场景 | 特点 |
|------|----------|------|
| 默认配置 | 日常使用 | 均衡设置 |
| 深度冥想 | 冥想时 | 更放松的背景 |
| 专注学习 | 学习工作时 | 提升专注力 |
| 助眠放松 | 睡前 | 更柔和，助眠 |

### 手动调参说明

说实话，**默认值就很好，不用改**。

但如果你想了解：

| 参数 | 默认值 | 范围 | 啥意思 |
|-----|-------|------|-------|
| 载波频率 | 17500 | 15000-20000 | 数字越大越"听不见" |
| 潜意识音量 | -23dB | -40~0 | 黄金值，别动 |
| 背景音乐音量 | 0dB | -20~10 | 背景太吵就调小 |
| Theta波 | 开启 | - | 帮你放松，推荐开 |
| 左耳频率 | 430Hz | 200-500 | Theta波基频 |
| 右耳频率 | 434Hz | 200-500 | 差频=右-左=4Hz |

## 常见问题

### 处理后的音频怎么"没声音"？
**正常！** 这就是要的效果。人声被搬到17500Hz以上了，你耳朵听不到，但大脑能接收到。

### 怎么验证它真的有效？
把载波频率改成14000，重新合成一次，你会听到"外星人说话"的声音。这就证明人声确实被处理了。

### 为什么要用WAV格式？
MP3会自动切掉16kHz以上的声音，你的潜意识信息就没了！所以必须用WAV。

### 多久能见效？
每天听1-2次，每次15-30分钟，坚持66天。用耳机效果更好。

### 批量处理怎么用？
上传时按住 Ctrl 多选文件，或多次拖拽上传。处理完成后可以一键下载全部。

### 预设怎么保存？
调整好参数后，点击"保存预设"，下次可以选择"自定义"加载。

## 录肯定语的小技巧

**✅ 好的说法：**
- "我越来越自信"
- "我值得被爱"
- "我每天都充满活力"

**❌ 不好的说法：**
- "我不自卑"（有负面词）
- "我没有焦虑"（大脑会忽略"没有"）

**录音建议：**
- 在安静的地方录
- 说清楚，语速适中
- 1-5分钟就够了
- 手机录音就行
