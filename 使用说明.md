# Subliminal Master - 潜意识音频合成引擎 使用说明

## 📁 软件简介

**Subliminal Master** 是一款专业的潜意识音频合成工具，基于神经科学和心理学原理，能够将您的肯定语转化为"听不见"的潜意识音频，并添加Theta波双耳搏动，帮助您更有效地进行潜意识编程和冥想。

### 核心逻辑

**隐藏轨（处理后的肯定句）+ 显性轨（背景音乐）+ 频率诱导（可选的双耳搏动）= 最终成品**

### 核心功能

- ✅ **双轨道输入**：支持上传肯定句音频和背景音乐
- ✅ **高频调制处理**：将人声调制到17.5kHz-19.5kHz，使其变得"听不见"
- ✅ **Theta波双耳搏动**：生成4Hz差频的双耳搏动，帮助放松和进入潜意识状态
- ✅ **音量控制滑动条**：精确控制潜意识轨、背景音乐和双耳搏动的音量
- ✅ **音轨对齐和循环**：自动对齐音轨长度，支持循环播放
- ✅ **无损WAV导出**：保证高频信息不被压缩损失
- ✅ **Web可视化界面**：在浏览器中操作，方便易用

## 🚀 快速开始

### 1. 启动应用

- 双击运行 `启动Subliminal_Master.bat` 文件
- 或者在命令行中运行：`python subliminal_master.py`

### 2. 打开浏览器

- 在浏览器中打开：http://localhost:5000
- 您会看到一个美观的拟态风格界面

### 3. 上传音频文件

#### Track A: 肯定句音频
- 点击或拖拽上传您的肯定句录音（TTS生成的语音）
- 支持格式：MP3, WAV, M4A, AAC, FLAC

#### Track B: 背景音乐
- 点击或拖拽上传冥想音乐、白噪音或大自然声音
- 支持格式：MP3, WAV, M4A, AAC, FLAC

### 4. 调整参数

#### 高频调制设置
- **载波频率 (Hz)**：建议17500-19500Hz
  - 将人声调制到此频率，使其变得"听不见"
  - 频率越高，越不容易被听到

- **潜意识轨音量 (dB)**：推荐值 -23dB（黄金值）
  - 过大会被听到，过小效果减弱

- **背景音乐音量 (dB)**：0dB = 原始音量
  - 根据需要调整背景音乐的音量

#### Theta波增强
- **开启 Theta 波增强**：推荐开启
- **左耳频率 (Hz)**：默认430Hz
- **右耳频率 (Hz)**：默认434Hz
- **差频**：4Hz Theta波
- **双耳搏动音量 (dB)**：推荐 -15dB

### 5. 开始合成

- 点击"开始合成"按钮
- 等待处理完成
- 下载处理后的音频文件

## ⚙️ 参数详解

### 1. 载波频率 (Hz)
- **默认值**：17500
- **建议范围**：17500-19500
- **作用**：将您的人声搬运到这个频率附近，使其变得"听不见"
- **原理**：使用振幅调制 (AM) 技术，将频谱搬移到载波频率 ± 原始频率的位置
- **调整建议**：频率越高，声音越不容易被听到，但也要考虑您的音频设备是否支持

### 2. 潜意识轨音量 (dB)
- **默认值**：-23
- **建议范围**：-30 到 -15
- **作用**：控制处理后的人声音量
- **调整建议**：
  - -23dB 是视频推荐的黄金值
  - 过大会被听到，过小效果减弱
  - 可以根据背景音乐的音量适当调整

### 3. 背景音乐音量 (dB)
- **默认值**：0
- **建议范围**：-10 到 10
- **作用**：调整背景音乐的音量
- **调整建议**：
  - 0dB = 原始音量
  - 根据潜意识轨的音量适当调整
  - 确保背景音乐不会掩盖潜意识信息

### 4. Theta波双耳搏动
- **左耳频率**：430Hz
- **右耳频率**：434Hz
- **差频**：4Hz Theta波
- **作用**：诱导大脑进入Theta波状态，增强潜意识接收效果
- **原理**：左右耳听到不同频率的声音，大脑会感知到差频
- **调整建议**：
  - Theta波 (4-8Hz) 有助于放松、冥想和潜意识接收
  - 可以根据需要调整差频

## ❓ 常见问题

### 1. 运行时提示缺少依赖库

**解决方案**：软件会自动安装缺少的依赖库，请耐心等待。如果安装失败，请手动运行以下命令：

```bash
pip install pydub numpy scipy flask
```

### 2. Web界面无法打开

**解决方案**：
- 确认命令行窗口显示"Running on http://localhost:5000"
- 检查防火墙是否阻止了Python
- 尝试在浏览器中输入 http://127.0.0.1:5000

### 3. 处理后的音频没有声音

**解决方案**：
- 这是正常现象！潜意识音频就是要"听不见"
- 如果想验证是否有效，可以：
  - 将载波频率降低到14000-15000Hz，会听到"外星人说话"般的声音
  - 使用频谱分析软件查看高频部分是否有信号

### 4. 处理后的音频文件很大

**解决方案**：
- WAV格式是无损的，文件较大是正常的
- 请勿转换为MP3，否则17.5kHz以上的高频信号会被删除
- 如果需要压缩，可以使用FLAC格式

### 5. 运行时出现FFmpeg警告

**解决方案**：这是正常现象，不影响软件运行。如果想消除警告，请下载并安装FFmpeg，然后将其添加到系统环境变量中。

### 6. 可以导出为MP3吗？

**答案**：**不可以！** MP3压缩算法会自动切掉16kHz以上的高频声音，如果切掉了，您的"潜意识信息"就真的消失了，只剩下安慰剂效应。必须使用WAV或FLAC格式。

## 🎓 科学原理

### 什么是潜意识音频？

潜意识音频（Subliminal Audio）是一种将肯定语或建议信息嵌入到音频中的技术，这些信息被调制到人类听觉阈值以下的频率，使其无法被有意识地听到，但可以被潜意识接收。

### 高频调制原理

使用振幅调制（AM）技术，将人声信号的频谱搬移到17.5kHz-19.5kHz的高频段。这个频率范围超出了大多数成年人的听觉范围，但内耳仍然可以接收到这些信号。

### 双耳搏动原理

双耳搏动（Binaural Beats）是一种听觉现象，当左右耳听到略有频率差异的声音时，大脑会感知到一个"差频"。例如，左耳听到430Hz，右耳听到434Hz，大脑会感知到4Hz的Theta波。

### Theta波的作用

Theta波（4-8Hz）是一种脑波状态，通常出现在深度放松、冥想和浅睡眠时。在这个状态下，大脑更容易接收和整合新的信息，是潜意识编程的理想状态。
